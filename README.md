# Solid Solidity

A full-stack Solidity + NextJS app that allows users to create custom Mechanical Keyboards on-chain. Tutorial by Pointer

## Installation

```bash
git clone https://github.com/brvangarcia/solid-solidity
Create .env.example from .env
source .env
yarn
```
